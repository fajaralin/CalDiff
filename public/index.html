📖 Teori Beda Hingga (Finite Difference)

Diferensiasi numerik adalah teknik untuk menghitung turunan suatu fungsi menggunakan nilai fungsi pada titik-titik diskret, tanpa perlu mengetahui ekspresi analitiknya secara eksplisit.

🟡 Forward Difference

Diturunkan dari ekspansi deret Taylor ke depan:

f(x+h) = f(x) + hf'(x) + h²f''(x)/2! + ...

Sehingga: f'(x) ≈ [f(x+h) - f(x)] / h

Error truncation: O(h) — proporsional terhadap h

🔴 Backward Difference

Diturunkan dari ekspansi deret Taylor ke belakang:

f(x-h) = f(x) - hf'(x) + h²f''(x)/2! - ...

Sehingga: f'(x) ≈ [f(x) - f(x-h)] / h

Error truncation: O(h) — proporsional terhadap h

🟢 Central Difference

Kombinasi forward dan backward, suku error orde pertama saling menghilangkan:

f'(x) ≈ [f(x+h) - f(x-h)] / 2h

Error truncation: O(h²) — jauh lebih akurat!
